Social Sharing  allows you to directly share your site links from a button link, which increase your site traffic from social sites.

== Installization ==
1. Upload the social-sharing.zip file from wp-admin panel.
2. Activate the plugin through plugin menu in wp-admin panel.
3. Enjoy it for free. :)
